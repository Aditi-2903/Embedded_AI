# Embedded Systems & AI: Constraints, Inference, and Integration

**Version 0.1 — Working Draft**
**Type:** Course Textbook — 15-week semester
**Target Reader:** Final-year undergraduate / entry-level graduate, ECE or CS
**Prerequisites:** C/C++, digital logic and microcontroller basics, signals and real-time concepts

---

> This book argues that AI deployment on embedded hardware is fundamentally a constrained optimization problem — and that the designer who cannot reason about both sides of that constraint will make integration decisions that fail in production.
