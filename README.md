# Embedded AI Book

A comprehensive book on Embedded Artificial Intelligence — covering the integration of AI and machine learning techniques into embedded systems and edge devices.

## About

This book explores how AI can be deployed on resource-constrained hardware, bridging the gap between machine learning theory and real-world embedded applications.

## Contents

- [Embedded_AI_Book.pdf](./Embedded_AI_Book.pdf) — Full book in PDF format

## Topics Covered

- Introduction to Embedded AI and Edge Computing
- Hardware platforms for embedded AI (microcontrollers, FPGAs, SoCs)
- Model optimization techniques (quantization, pruning, knowledge distillation)
- Deploying ML models on embedded devices
- Real-world embedded AI applications
